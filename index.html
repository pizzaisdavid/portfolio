Tell Me Timer

Ever realize you have no idea how long you've been standing in the shower, cycling, or working out?


TellMeTimer has your back. Just set reminder frequency, press start, and you'll receive verbal reminders for the current time until the app is turned off. For those who like to listen to music awhile working out, don't worry TellMeTimer is music friendly, it pauses songs during reminders and unpauses them afterwards.

Available on the Google Play Store





Technology Used
  • Java
  • Android Studio

Pocket Stats

I used to use an app called Pocket that allows users to save articles to read for later. It is a cool application, but they don't give you any statistics.


This lead me to create Pocket Stats, which tells the user the number of articles they have saved. The user logins via OAuth 2.0 to receive an access token and then a request is automatically made to the Pocket servers to fetch all their saved articles title which are then tallied up.



Watch the concept video.





Technology Used
  • Java
  • Android Studio
  • OAuth 2.0
